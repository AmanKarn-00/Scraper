# Nepal Stock Market Scraper

A Python web scraper that extracts real-time stock market data from Nepal Stock Exchange (NEPSE) and saves it to CSV format.

## Features

- Extracts stock data from Merolagani website
- Parses stock information including:
  - Stock Symbol
  - Last Traded Price (LTP)
  - Percentage Change
  - Open, High, Low prices
  - Trading Quantity
- Saves data to CSV file for further analysis
- Handles 339+ stocks automatically
